# 📰 全球新闻简报 — 自动抓取推送

每2小时自动抓取全球重要新闻，按 **政治/经济/军事/科技/社会** 五大分类整理，翻译、核实后推送到飞书。

## 核心能力

- **29+ 信源**：国际通讯社（Reuters/BBC/AP）+ 中国官媒（新华社/央视/环球网）直连抓取
- **自动翻译**：海外英文信源通过 LLM 自动翻译为中文
- **交叉核实**：跨源相似报道聚类，多源报道标记为已核实
- **存疑标注**：单源低可信度、敏感词、标题夸张自动标注 ⚠️
- **智能评分**：信源权重 + 关键词匹配 + 中国相关 + 突发新闻多维评分

## 信源覆盖

| 分类 | 来源 |
|------|------|
| 国际通讯社 | Reuters / AP / AFP / BBC / NPR |
| 国际主流 | CNN / NYT / Guardian / Al Jazeera |
| 中国官媒 | 新华社 / 央视 / 人民网 / 环球网 / 澎湃 / 财新 / 中新网 |
| 财经 | Bloomberg / FT / 界面新闻 |
| 热搜聚合 | 微博 / 知乎 / HackerNews / 36氪 |

## 文件结构

```
├── SKILL.md       # 使用文档
├── briefing.py    # 主引擎（抓取/评分/翻译/验证/推送）
├── cn_sources.py  # 中国信源直连抓取器（13源）
└── config.json    # 信源配置 / 分类关键词 / 评分参数
```

## 快速使用

```bash
# 运行一次
python3 briefing.py

# 试运行（不发送）
python3 briefing.py dry
```
